# Qraft Roadmap

> This roadmap reflects current thinking and is subject to change. Community feedback drives priorities — open an issue or discussion to influence what comes next.

## v0.1.x — Stabilization (current)

- [x] Config validation with clear error messages
- [x] PyPI publication of `qraft` and `qraft-utils`
- [x] Documentation and migration guides
- [x] `.pyi` type stubs for IDE support

## v0.2.0 — Core Gaps

- [ ] Seeds: load reference data from CSV files into tables
- [ ] `--vars` CLI override: pass variables from the command line
- [ ] Source freshness monitoring
- [ ] Singular data tests (standalone SQL test files)
- [ ] Improved error messages and diagnostics

## v0.3.0 — Developer Experience

- [ ] `qraft retry`: re-run only failed models from the last run
- [ ] `qraft list`: list models, sources, tests
- [ ] Watch mode for development
- [ ] Performance benchmarks and optimization
- [ ] Pre-commit hook integration
